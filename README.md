# Golden Dragon Restaurant

A premium Pan-Asian restaurant website featuring beachfront dining at Al Hira Beach, Sharjah, UAE.

## Live Demo

[View the live site](https://yourusername.github.io/goldendragon)

## Features

- **Hero Section** - Stunning beachfront dining imagery with call-to-action buttons
- **Experience Section** - Showcase the beach dining atmosphere
- **Cuisine Section** - Highlight authentic Pan-Asian flavors
- **Menu Section** - Three category cards (Sushi, Chinese, Buffet)
- **Sushi Bar & Buffet Sections** - Full-screen feature sections
- **Reviews** - Customer testimonials
- **Private Dining** - Event hosting showcase
- **Gallery** - Restaurant interior and food photography
- **Reservations** - Booking form with contact options
- **Contact/Footer** - Location, hours, and social links

## Design

- **Color Palette**: Dark luxury theme with gold accents
  - Background: #0B0B0D (near-black)
  - Accent: #C69B3C (antique gold)
  - Text: #F4F1EA (warm ivory)
- **Typography**: Playfair Display (headings), Inter (body), Space Grotesk (labels)
- **Responsive**: Fully mobile-friendly design

## GitHub Pages Setup

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to Settings > Pages
4. Select "Deploy from a branch" and choose "main"
5. Your site will be live at `https://yourusername.github.io/repository-name`
